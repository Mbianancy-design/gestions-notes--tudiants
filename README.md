# 📚 Gestion des Notes des Étudiants - Laravel

Application web de gestion des notes des étudiants, développée avec Laravel dans le cadre de ma formation BTS Informatique.

## 📌 Description

Cette application permet de gérer les notes des étudiants via une interface web complète, avec une base de données relationnelle MySQL pour stocker et organiser les informations (étudiants, matières, notes).

## ⚙️ Fonctionnalités

- Utilisation des routes et contrôleurs (architecture MVC)
- Gestion CRUD complète (ajout, modification, suppression et consultation des notes)
- Interface utilisateur avec le moteur de templates Blade
- Conception de la base de données (MCD/MLD) et requêtes SQL associées

## 🛠️ Technologies utilisées

- Laravel (PHP)
- MySQL
- Blade
- HTML / Boostrap

## 📚 Ce que ce projet m'a appris

- Architecture MVC et organisation d'un projet Laravel
- Conception de base de données relationnelle
